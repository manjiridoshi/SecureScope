# Security Policy

## Responsible Use

SecureScope is designed for **authorized security testing only**. Users must:

- Obtain explicit permission before scanning any systems
- Comply with all applicable laws and regulations
- Use the tool ethically and responsibly
- Not use it for malicious purposes

## Reporting Security Issues

If you discover a security vulnerability in SecureScope itself:

1. **Do NOT** open a public issue
2. Email the maintainer directly with details
3. Allow reasonable time for a fix before public disclosure

## Disclaimer

This tool is provided "as is" without warranty. The authors are not responsible for any misuse or damage caused by this tool.

## Legal Notice

Unauthorized scanning of systems you don't own may be illegal in your jurisdiction. Always ensure you have proper authorization before conducting security assessments.

## Best Practices

When using SecureScope:

- Always get written permission
- Respect rate limits and system resources
- Document your testing activities
- Report findings responsibly
- Follow coordinated disclosure practices

Thank you for using SecureScope responsibly!
